# Ape Banana Collection Simulation

A multi-threaded simulation of ape families collecting bananas in a maze, with real-time OpenGL visualization.

## Features
- **Multi-threaded Architecture**: Each family member runs in separate threads
- **Real-time Graphics**: OpenGL visualization of maze and agents
- **Complex Behaviors**: 
  - Females search for bananas and fight when returning
  - Fathers protect baskets and fight with neighbors
  - Babies steal bananas when hungry
- **Configurable Parameters**: All simulation parameters in config.txt
- **Random Maze Generation**: Dynamic maze creation with obstacles
